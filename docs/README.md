# Docs

AutoFlow 的框架文档统一维护在 `docs/` 下，仅记录框架能力、约定、格式与边界；插件的业务/实现文档放在各插件目录内。

## 选择语言

- 中文文档入口：[`docs/zh/README.md`](zh/README.md)
- English docs index: [`docs/en/README.md`](en/README.md)

## 文档范围

- 允许：Flow/TriggerDoc/Runner/日志与产物/Secrets/插件系统（SDK）等框架级约定
- 不包含：任何具体插件的业务流程与实现细节（放到 `plugins/<plugin>/`）
